# 安全说明

AI Dream 当前定位为本地开发和学习环境，但仍应遵守基本安全规则。

## 本地密钥规则

- 不要提交 `.env`、数据库密码、API Token、私钥、模型服务凭据或任何真实用户数据。
- 使用 `.env.example` 记录变量名称和本地默认值，真实值只保存在本机环境中。
- 本地生成的 `artifacts/`、`datasets/`、数据库目录和 Docker 状态目录不应进入版本控制。

## Prompt 与容器安全

- 不要在 Prompt、日志、测试快照或演示数据中写入真实密钥、个人身份信息或商业敏感内容。
- 不要把宿主机密钥文件挂载进容器，除非已经确认最小权限和只读访问。
- 不要在镜像层中写入密钥；构建参数、环境变量和缓存都可能泄露。

## 报告问题

如果发现安全问题，请先私下联系维护者并提供复现步骤、影响范围和建议修复方式。不要在公开 issue 中暴露可利用细节或真实凭据。
